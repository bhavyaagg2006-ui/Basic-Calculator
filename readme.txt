Basic Calculator

A modern, stylish, and fully functional calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations along with extra advanced features and a beautiful animated UI.


---

Features

Basic Arithmetic: Addition, subtraction, multiplication, division.

Advanced Operations:

Square root (√)

Square (x²)

Percentage (%)

Sign toggle (±)


Memory Functions:

M+ (Add to memory)

M- (Subtract from memory)

MR (Memory recall)

MC (Memory clear)


Keyboard Input Support: Type numbers and operators directly.

Error Handling: Detects division by zero and invalid calculations.

Theme Toggle: Light and dark theme switch.

Modern UI/UX:

Gradient glowing buttons

Glassmorphism effect

Smooth hover and click animations

Responsive design for desktop & mobile

How to Use

1. Download or clone this repository:

git clone https://github.com/bhavyaagg2006-ui/Basic-Calculator.git

2. Open the index.html file in your browser.


3. Use the buttons or your keyboard to perform calculations.


4. Click C to clear the screen or use MC/MR/M+/M- for memory operations.


5. Toggle between Light/Dark themes using the top-right switch.

Error Handling

Division by zero displays an error message.

Invalid expressions are caught using try...catch.

Memory operations work only on valid numbers.
